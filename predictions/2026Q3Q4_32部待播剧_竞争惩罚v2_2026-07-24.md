## 〇、v2.0 改进说明

### 新增：同期同赛道竞争惩罚

```
comp_penalty = 1 - density × (1 - hot_index / 10)
new_hot_index = hot_index × comp_penalty
```

| 参数 | 说明 |
|------|------|
| density | 同月同赛道节目数量决定的竞争密度 |
| hot_index | 原六维度加权得分 |
| comp_penalty | 竞争惩罚系数 |

> **逻辑：同月同赛道节目越多、自身分数越低，惩罚越重。** 强者份额大，被蚕食比例小；弱者份额小，在拥挤赛道中更难突围。
